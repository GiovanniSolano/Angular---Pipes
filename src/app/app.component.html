# Nombre Descripción Uso
1 Currency

Tranforma un número en una cadena de moneda de acuerdo a reglas de configuración (región, maximo de números enteros, decimales - min y max)

 {{precio | currency:'USD':'symbol':'1.1-3'}}
2 Decimal

Transforma un número para que aparezca con el número de decimales que queramos y con el número de digitos en la parte entera que se requieran

 {{numero | number:'1.2-4'}}
3 Date

Formateo de fechas de acuerdo a lo requerido

{{fechaActual | date: 'dd-MM-yyyy HH:mm:ss'}}
4 Percent

Transformar un número a su representación en porcentaje

{{numAleatorio | percent:'1.1-2'}}
5 JSON

Obtener la representación de un objeto en formato string

 {{estudiante | json}}
6 UpperCase

Tranformar un texto a mayúsculas

 {{texto | uppercase}}
7 LowerCase

Tranformar un texto a minúsculas

 {{texto | lowercase}}
8 TitleCase

Capitalizar un texto (primera letra de cada palabra en mayúsculas)

 {{texto | titlecase}}
8 Slice

Tomar los elementos que nos interese de un array o la parte en caso de un string
9 Async

Permite resolver una promesa directamente dentro de nuestro HTML

 Mensaje: espere 4 seg...{{promesa | async}}
10 Custom pipe (Invertir string)

Pipe personalizado para invertir un string

 {{texto | reverso}}
11 Custom pipe (Capitalize)

Pipe personalizado para capitalizar cada palabra de una oración

 {{texto | capitalize}}
12 Custom pipe (Num to Array)

Pipe personalizado para tomar un número cualquiera (sin tener que ser una colección ya definida) y convertirlo en un array para utilizar un ngFor
  • {{num}}
